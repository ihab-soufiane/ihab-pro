Student Management System

A complete web system designed to manage student information, including registration, data updates, and profile consultations, tailored for educational institutions.

Technologies Used

  • Backend: PHP
  • Database: MySQL
  • Frontend: HTML, CSS
  • Environment: Local server with XAMPP

    • Key Features

  • Student Management: Add, edit, and delete student profiles
  • Data Consultation: View detailed information about registered students.
  • User-Friendly Interface: Simple and intuitive web pages.
  • Data Security: Used prepared statements to prevent SQL injection.
